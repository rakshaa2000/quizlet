Installation
============


.. note::
   Only Python 3 is supported.


From source
-----------

.. code-block:: bash

   $ git clone https://github.com/s-alexey/quizlet.git
   $ cd quizlet
   $ python setup.py

After installing you can access `quizlet` module:

.. code-block:: bash

    $ python -c "import quizlet; print(quizlet.__version__)"
    0.1
